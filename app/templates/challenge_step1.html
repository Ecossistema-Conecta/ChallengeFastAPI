Etapa 1: Começando sua Jornada

Bem-vindo ao primeiro passo do desafio! Nesta etapa, você deverá criar rotas básicas para um CRUD de super-heróis. Você já possui a tabela e os schemas relacionados. A missão é simples: utilizar o FastAPI em conjunto com o SQLAlchemy para inserir, listar, atualizar e remover heróis do banco de dados.

O que você precisa fazer?

  • Criar rotas como /heroes para listar, /heroes/{id} para detalhar, POST /heroes para criar e DELETE /heroes/{id} para remover.
  • Utilizar os schemas Pydantic já definidos para validação de entrada/saída.
  • Trabalhar com o Depends do FastAPI para gerenciar a sessão do banco (SQLAlchemy).

Resumo sobre FastAPI e CRUD

O FastAPI é um framework rápido e moderno, ideal para construir APIs utilizando Python. Ele é intuitivo e possui tipagem forte, o que facilita a manutenção e testes. Para trabalhar com banco de dados, geralmente utiliza-se o SQLAlchemy e o Alembic (para migrações). O padrão é criar um objeto Session através do `Depends`, facilitando a injeção de dependência.

Os schemas (modelos Pydantic) garantem que a entrada e a saída das rotas estejam de acordo com o esperado, enquanto as models do SQLAlchemy representam as tabelas do banco.
