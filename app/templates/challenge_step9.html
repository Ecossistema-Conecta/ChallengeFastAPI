Etapa 9: Templates e Design Thinking para Wiki de Heróis

Nesta etapa, você criará páginas detalhadas para cada herói, no estilo de wikis como as de Dota ou LoL. O foco será criar uma interface atraente e informativa, com elementos visuais que realcem os atributos e habilidades dos heróis.

O que você vai fazer?

  • Desenhar um protótipo no Figma ou outra ferramenta similar.
  • Estruturar o front end com o Jinja2 ou outro framework escolhido (Vuejs, React, etc).
  • Adicionar elementos visuais como ícones, gráficos e tabelas estilizadas.

Por que Design Thinking?

Design Thinking ajuda a criar interfaces que priorizam a experiência do usuário (UX). Ao prototipar antes de implementar, você evita retrabalho e garante que a interface final esteja alinhada com as necessidades dos usuários.

Usando o Figma para Prototipar

O Figma é uma ferramenta poderosa para criar protótipos interativos. Você pode desenhar os layouts das páginas, testar interações e obter feedback antes de implementar. 


# Estrutura típica de uma página de herói
- Nome do Herói
- Imagem de destaque
- Atributos principais (força, velocidade, etc.)
- Habilidades (com descrição e efeitos)
- Histórico

Estruturando a Tela

Você pode fazer o front end dentro do projeto do FastAPI com Jinja2 ou criar seu front end com frameworks especializados, como VueJs ou React.
